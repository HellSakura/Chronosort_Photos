# ChronoSort Photos (照片时间排序工具)

一个根据照片的EXIF日期或文件修改日期来重命名它们的Python脚本。

## 功能

*   **日期检测**:   
按以下顺序优先级排序：
    1.  EXIF "拍摄日期"
    2.  文件修改日期
    3.  文件创建日期
*   **后续处理**:
    *   添加数字后缀，解决同日期文件名冲突。
    *   自动跳过已经符合格式的文件。

## 使用方法

1.  **安装依赖**
    ```bash
    pip install Pillow
    ```
2.  **运行脚本**
    ```bash
    python chronosort_photos.py
    ```
3.  **选择文件夹**
4.  **检查预览**  
查看终端中打印出的修改列表。
5.  **确认操作**    
 输入 `yes` 并按回车键以应用更改。
